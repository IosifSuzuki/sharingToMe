About project

Hi, my name is Bogdan, just i tell you about the project. It's project for store audio, especially podcast. In future we'll support other media type. The web version is only for upload file. Main application will be in iphone, he'll support background play (screen off) a podcast during you prepare to sleep, do housework, sit in a bus or something another. Main goal of the application is download audio to external storage from remote server and play media files in offline mode. After achieve a main goal, we'll add a new additional features like a chat in real time, mini games, expand existing functional. You'll see a development process bellow in table view.
